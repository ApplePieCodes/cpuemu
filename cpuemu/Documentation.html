Scythe Instruction Set

The Scythe CPU Architecture Instruction Set

[IMM(size): usage]: Immediate Arguement
[REG: usage]: Register Value Arguement

movb [IMM(byte): source] [REG: destination] (0x00)

Moves [source] to first byte in [destination].

movb [REG: source] [REG: destination] (0x01)

Moves first byte of [source] to first byte in [destination].

movw [IMM(word): source] [REG: destination] (0x02)

Moves [source] to first word in [destination].

movw [REG: source] [REG: destination] (0x03)

Moves first word of [source] to first word in [destination].

movd [IMM(dword): source] [REG: destination]

Moves [source] to first dword in [destination].

movd [REG: source] [REG: destination]

Moves first dword of [source] to first dword in [destination].

movq [IMM(qword): source] [REG: destination]

Moves [source] to [destination].

movq [REG: source] [REG: destination]

Moves [source] to [destination].
